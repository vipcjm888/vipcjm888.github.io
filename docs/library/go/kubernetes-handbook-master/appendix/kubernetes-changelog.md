# Kubernetes版本更新日志

刚开始写作本书时 Kubernetes1.6 刚刚发布，随后 Kubernetes 基本以每三个月发布一个版本的速度不断迭代，为了追踪不同版本的新特性，我们有必要在此记录一下。

每个 Kubernetes 版本的详细更新日志请参考：https://github.com/kubernetes/kubernetes/blob/master/CHANGELOG.md
